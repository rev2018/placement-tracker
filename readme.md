# Placement Tracker (Full Stack Web App)

A full-stack placement tracker to manage job applications and hiring stages.

## Features
- Signup / Login / Logout
- Add / Edit / Delete job applications (CRUD)
- Track status: Applied → Test → Interview → Selected/Rejected
- Search by company or role
- Filter by status
- Dashboard counts
- Export applications as CSV

## Tech Stack
- Python (Flask)
- HTML, CSS, JavaScript
- SQL (SQLite)
- Git/GitHub
